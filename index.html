🌐 Share the Viewer Version Online (Free)
Option A — Netlify Drop (easiest, 30 seconds)
1. Save this .html file to your computer
2. Go to netlify.com/drop
3. Drag the file onto the page → instant permanent URL
4. Share that URL with your team
5. When you have new weekly data: re-process, re-drag the updated file

⚡ Tip: Switch to Viewer mode before sharing — the URL opens in whatever mode was last active.
Option B — GitHub Pages (auto-updates)
1. Create a free GitHub account → New repository → upload this .html as index.html
2. Go to Settings → Pages → Source: main branch
3. GitHub gives you a permanent URL like yourname.github.io/repo
4. Each week: replace the file and commit — viewers see updated data instantly
